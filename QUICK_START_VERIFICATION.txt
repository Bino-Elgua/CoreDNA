PROVIDER ROUTING:
================================================================================

TEXT EXTRACTION (Brand DNA, Leads, etc):
  → Uses: settings.llms[activeLLM]
  → Function: getActiveLLMProvider()
  → Providers: Google, OpenAI, Claude, Mistral, Groq, DeepSeek, etc.

IMAGE GENERATION (Assets, etc):
  → Uses: settings.image[activeImageGen]
  → Function: generateAssetImage()
  → Providers: DALL-E, Stable Diffusion, Flux, Midjourney, etc.

VIDEO GENERATION (Future):
  → Uses: settings.video[activeVideo]
  → Providers: Sora 2, Veo 3, Runway, Kling, etc.

VOICE/TTS:
  → Uses: settings.voice[activeVoice]
  → Providers: ElevenLabs, OpenAI TTS, PlayHT, etc.
